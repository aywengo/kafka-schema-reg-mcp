## üîê OAuth 2.1 Universal Configuration


## üì¶ Tools to Resources Migration

### **Breaking Change: Tools Migrated to Resources**

Several tools have been migrated to MCP resources for better performance and caching:

| **Old Tool** | **New Resource** | **Migration Required** |
|-------------|------------------|----------------------|
| `list_subjects` | `registry://{name}/subjects` | ‚úÖ **YES** |
| `list_registries` | `registry://names` | ‚úÖ **YES** |
| `list_contexts` | `registry://{name}/contexts` | ‚úÖ **YES** |
| `get_schema` | `schema://{name}/{context}/{subject}` | ‚úÖ **YES** |
| `get_schema_versions` | `schema://{name}/{context}/{subject}/versions` | ‚úÖ **YES** |
| `get_global_config` | `registry://{name}/config` | ‚úÖ **YES** |
| `get_subject_config` | `subject://{name}/{context}/{subject}/config` | ‚úÖ **YES** |
| `get_mode` | `registry://mode` | ‚úÖ **YES** |
| `get_subject_mode` | `subject://{name}/{context}/{subject}/mode` | ‚úÖ **YES** |

### **Migration Examples**

#### **List Subjects Migration**
```python
# ‚ùå OLD WAY (No longer works)
result = await client.call_tool("list_subjects", {"context": "production"})

# ‚úÖ NEW WAY (Use resources)
result = await client.read_resource("registry://default/subjects")
# or for specific registry:
result = await client.read_resource("registry://production/subjects")
```

#### **List Registries Migration**
```python
# ‚ùå OLD WAY (No longer works)
result = await client.call_tool("list_registries", {})

# ‚úÖ NEW WAY (Use resources)
result = await client.read_resource("registry://names")
```

#### **Get Schema Migration**
```python
# ‚ùå OLD WAY (No longer works)
result = await client.call_tool("get_schema", {
    "subject": "user-events",
    "context": "production"
})

# ‚úÖ NEW WAY (Use resources)
result = await client.read_resource("schema://default/production/user-events")
# or for specific registry:
result = await client.read_resource("schema://prod-registry/production/user-events")
```

### **Response Format Changes**

Resources return data in a different format than tools:

```python
# Tool response (old)
if result.content and len(result.content) > 0:
    data = json.loads(result.content[0].text)

# Resource response (new)
if result.contents and len(result.contents) > 0:
    data = json.loads(result.contents[0].text)
```

### **Benefits of Resources**

- üöÄ **Better Performance**: Resources can be cached by MCP clients
- üìä **Structured Data**: More predictable data format
- üîÑ **Real-time Updates**: Resources can be updated without tool calls
- üí∞ **Lower Token Usage**: Reduced overhead for LLM interactions
